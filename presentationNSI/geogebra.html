k plus proches voisins

Le thème des k plus proches voisins est le thème permettant de toucher du doigt celui de l'intelligence artificielle.
Il s'agit simplement, à partir de données, de déterminer une valeur pour un nouvel entrant dans la base des données de départ.
Exemple : un nouveau joueur dans une équipe de rugby.
