HTML, CSS ,JS

En spécialité NSI on pourra voir au cour de l'année un language appeler HTML. Le language HTML est un language se présentant sous forme de balises ouvrantes et de balises fermantes, grâce à ces balises on pourra créer des pages web et des sites web comme celui-ci. Pour rajouter du style à une page HTML on pourra utilisé un autre language imformatique se nommant CSS qui nous permettera donc de mettre en forme la page web. Enfin on utilisera un dernier language, le JAVA SCRIPT qui nous permettera de rajouter des fonctions avec des variables, ect... dans une page web afin de créer des éléments interactives dans cette même page.

En nsi nous découvrons ce langage sans approfondir et on à donc pas besoin d'apprendre toutes les balises ou toutes les propriétées de chaque languages.

BALISES

Parmis les balises que l'on peut voir en HTML durant l'année de NSI il y a par exemple :

  • «p» «/p» : Servant à écrir un paragraphe sur la page.
  • «b» «/b» : Permettant de mettre en Gras des characters.
  • «a» «/a» : Permet de créer des liens hypertexte.
On pourrat aussi voir des commandes en CSS comme par exemple :
  • color: blue; : Qui permet par exemple de changer la couleur d'un texte
  • text-decoration: underline; : Cela permet de souligner un texte

JAVA SCRIPT

Le java script nous permet d'intégrer du code dans le HTML Comme par exemple une fonction créant une alerte à l'écran
On peut aussi faire des calcules grâce à java script comme des additions
