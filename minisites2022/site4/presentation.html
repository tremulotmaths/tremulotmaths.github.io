    Qu'est-ce que la spécialité NSI ?

La spécialité NSI permet aux élèves de pratiquer languages de programmation afin de comprendre le fonctionnement de certains logiciels ou encore des sites internet.

En NSI, nous utilisons principalement le language python qui est un des languages les plus utilisé en programmation. Nous pouvons aussi retrouver les languages HTML, CSS et Javascript qui serviront à crée une page web
Les élèves utilisent ses notions afin de réaliser des exercies d'entrainement ou des projets, ces projets sont des exercices qui permettent aux élèves de manipuler les différents languages et de les comprendre
