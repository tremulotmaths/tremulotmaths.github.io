Les différents langages de programmation étudiés en 1ère NSI




Python

Python est le langage de programmation choisi par l'état pour être enseigné, et ce pour plusieurs raisons:
  • Il est simple à apprendre et à utliser de par sa syntaxe peu exigeante, claire et lisible.
  • Python est un langage polyvalent. Il est utilisé dans de nombreux domaines comme les jeux vidéos, l'intelligence artificelle, l'analyse de données et plus encore.
  • C'est un langage complet grâce à ses nombreuses bibliothèques comme maths, mathsplotlib ou encore random ce qui lui permet de figurer parmi les langages de programmation les plus populaires du monde.

HTML

Le langage HTML(Hypertext Markup Langage) est un langage de balisage utilisé pour créer des sites web, il permet de créer la structure du site internet. C'est un des langages étudié au cours de l'année de première spéclialité NSI.

CSS

Le CSS, ou Cascading Style Sheets, est un langage de feuilles de style utilisé pour définir la présentation et l'apparence visuelle des pages web, notamment la couleur, la police de caractères, la mise en page et les effets visuels. Il permet de séparer la structure HTML du style, ce qui facilite la gestion et la maintenance des sites web. Le CSS fonctionne en cascade, ce qui signifie que les règles peuvent être spécifiées à différents niveaux et se combiner pour déterminer le style final d'un élément.

Java Script

JavaScript (JS) est un langage de programmation de script utilisé principalement pour rendre les pages web interactives et dynamiques en manipulant le contenu HTML et en réagissant aux actions des utilisateurs. Il peut être exécuté côté client dans le navigateur web ou côté serveur avec des environnements comme Node.js.
