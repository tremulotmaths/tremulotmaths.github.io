Probability using Mathigon

Warm up

Predict the number of times you'll get all the coins ending up the same (either all HHH or all TTT), when you flip all three coins 100 times.

Select the three coins and click on "Flip" to know how it works.

The main activity

Using the Polypad, flip 25 sets of 3 coins a total of 8 times. Each time, update the total number of HHH and TTT flipped and the percentage of each.
