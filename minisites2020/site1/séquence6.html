Séquence 6 : Interfaces Systèmes / Shell

Dans cette séquence, nous avons abordé les interfaces systèmes. Nous avons commencé par apprendre ce qu'est le STANDARD POSIX. La plupart des systèmes d'exploitation sont compatibles à l'exception de Windows. Par la suite, nous avons défini des notions comme noyau ou logiciel libre par exemple.

Nous avons également défini le Shell. C'est une forme simple d'interface utilisateur. C'est le terminal pour faire simple. Le Shell était l'interface principale avant que les avancées technologiques permettent d'interagir à l'aide d'interfaces graphiques
