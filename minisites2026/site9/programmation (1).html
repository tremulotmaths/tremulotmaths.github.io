CODE La programmation en NSI

En NSI, on apprend à programmer avec plusieurs langages selon les besoins. Chaque langage a un rôle précis.

Python

Langage principal. On l'utilise pour les algorithmes, le traitement de données et les projets.

HTML / CSS

Pour créer des pages web — structure avec HTML, mise en forme avec CSS.

JavaScript

Pour rendre les pages web interactives — boutons, animations, formulaires.

SQL

Pour interroger des bases de données — sélectionner, filtrer, trier des données.
