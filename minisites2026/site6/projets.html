🐍 Python - Les bases

Python est le langage principal enseigné en NSI. Voici les concepts fondamentaux:

Les Variables

En Python, les variables stockent des données. On définit une fonction avec def:

# Exemple de fonction avec variable:

def calcul(a, b):
    c = a + b
    print(c)

# Explication:

  • def calcul(a, b): - Déclare une fonction avec deux paramètres
  • c = a + b - Crée une variable qui stocke la somme
  • print(c) - Affiche le résultat
